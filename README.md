# 📱 Solicitudes Contractuales App

Aplicación móvil desarrollada en **Flutter** para la gestión de solicitudes contractuales, permitiendo a los usuarios registrar, consultar y administrar procesos de manera eficiente y organizada.

---

## 🚀 Descripción

**Solicitudes Contractuales App** es una solución digital orientada a optimizar la gestión de solicitudes dentro de una organización. La aplicación permite centralizar la información, mejorar el seguimiento de procesos y facilitar la interacción entre usuarios y administradores.

---

## 🛠️ Tecnologías utilizadas

* Flutter
* Dart
* API REST (Backend)
* Git & GitHub

---

## 📋 Características principales

* 📌 Registro de solicitudes contractuales
* 📊 Consulta de estado de solicitudes
* 🔐 Autenticación de usuarios
* 📂 Gestión de información centralizada
* ⚡ Interfaz rápida y responsiva

## ⚙️ Instalación y ejecución

### 1. Clonar el repositorio

```bash
git clone https://github.com/cryger/solicitudes_contractuales.git
```

### 2. Ingresar al proyecto

```bash
cd solicitudes_contractuales
```

### 3. Instalar dependencias

```bash
flutter pub get
```

### 4. Ejecutar la aplicación

```bash
flutter run
```
